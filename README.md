# 🕵️ ShadowAI — AI-Powered Security Intelligence

**Created by: Ankit Patidar**

ShadowAI is an AI-powered security assistant that answers ANY question about cybersecurity, hacking, networking, programming, and general knowledge.

## 🚀 Live Demo
```
https://shadowai-mkgu.onrender.com
```

## 🎯 Features
- 💬 **ChatGPT-style interface** — Ask anything!
- 🕵️ **Security focus** — Hacking, CTFs, bug bounty
- 🌐 **General knowledge** — Pizza recipe, technology, etc.
- 🔄 **Multi-provider AI** — Groq, Gemini, OpenAI
- 📱 **Mobile responsive** — Works on phone too!

## 🛠️ Tech Stack
- Python + Flask (backend)
- HTML/CSS/JS (frontend)
- Groq/Gemini/OpenAI API
- Render (hosting)

## 🚀 Local Setup
```bash
git clone https://github.com/ANKIT48274/shadowai.git
cd shadowai

# Set API key (any one)
export GROQ_API_KEY=gsk_your_key
# OR
export GEMINI_API_KEY=your_key

# Install & run
pip install flask gunicorn
python3 app.py
```

## 🌍 Deploy to Render
1. Fork repo to GitHub
2. Create Web Service on Render
3. Add `GROQ_API_KEY` env var
4. Deploy! 🚀
